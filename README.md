## How to Compile and Run
In the terminal, run

$ make

to compile all files.

To compile only parts of the code, you can specify these options: 

solve_test, deriv_test, and/or extremum_test

Then, to test the scripts, run

$ ./solve_test
$ ./deriv_test
$ ./extremum_test
